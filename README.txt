GA Power Dashboard — Polished Web Build (Georgia Power Theme)

WHAT'S INCLUDED
- index.html  → full interactive dashboard (autoplay → manual slider, per-scenario overviews, Monte Carlo, AI, Policy)

SHARE AS A LINK (RECOMMENDED)
1) Upload index.html to a new GitHub repository (e.g., ga-power-dashboard).
2) In GitHub: Settings → Pages → "main branch / root" → Save.
3) Your public link will be: https://<yourname>.github.io/ga-power-dashboard/

OFFLINE / TEXTING OPTION
- Open index.html in Chrome → File → Save Page As → "Webpage, Complete".
- This creates an .html + a folder of assets. Zip them both before sharing.

MOBILE VIEW TIP
- On iPhone/Android, open the link in Safari/Chrome. Texting the raw .html can break charts—send the link.

THEME
- Georgia Power palette: navy #002A5C, red #E41C23, blue #0076BF, aqua #00AEEF, slate #444F59.
- Background: #F8F9FA. Fonts: Segoe UI/Helvetica (system fonts).
